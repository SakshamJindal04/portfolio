## 🚀 Overview

This repository contains the source code for my personal developer portfolio. The goal of this project is to create a fast, accessible, and beautifully designed platform to present my journey in software development.

### ✨ Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop devices.
- **Project Showcase:** Detailed views of my latest work, including links to live demos and source code.
- **Interactive UI:** Smooth animations and transitions.
- **Fast Performance:** Built on Next.js for server-side rendering and optimal load times.

## 🛠️ Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SakshamJindal04/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn install / pnpm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or yarn dev / pnpm dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
